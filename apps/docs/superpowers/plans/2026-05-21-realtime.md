# Realtime Extension Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship a cleanroom realtime subsystem for shadmin: a `RealtimeDataProvider` extension, four transports (WebSocket / SSE / BroadcastChannel / in-memory fake), full hook surface (`useSubscribe`, `usePublish`, `useGetListLive`, `useGetOneLive`, `useGetManyLive`, lock hooks), and drop-in component wrappers (`<ListLive>`, `<EditLive>`, `<ShowLive>`, `<MenuLive>`, `<LockOnMount>`, `<LockStatus>`). Lives entirely in a new `src/components/realtime/` feature folder.

**Architecture:** Transport-as-plugin design — `realtimeDataProvider(baseDP, transport, { locks? })` wires a `RealtimeTransport` into an ra-core `DataProvider`. Transports own their connection lifecycle (lazy connect, idle-disconnect, backoff reconnect). Hooks invalidate react-query caches on incoming events. Components are thin wrappers around existing admin views + a subscription effect. No `any` in our own code. Zero references to upstream proprietary code.

**Tech Stack:** TypeScript (strict), React 19, ra-core 5.14, @tanstack/react-query 5.83, vitest + vitest-browser-react + Playwright (Chromium), pnpm 10.x.

**Spec:** [docs/superpowers/specs/2026-05-21-realtime-design.md](../specs/2026-05-21-realtime-design.md)

import type { DataProvider, Identifier } from "ra-core";

export type RealtimeEventType =
  | "created"
  | "updated"
  | "deleted"
  | "locked"
  | "unlocked"
  | (string & {});

export interface RealtimeEvent<P = unknown> {
  topic: string;
  type: RealtimeEventType;
  payload: P;
  meta?: Record<string, unknown>;
}

export type SubscriptionCallback<P = unknown> = (event: RealtimeEvent<P>) => void;
export type Unsubscribe = () => void;

export interface Lock {
  resource: string;
  recordId: Identifier;
  identity: Identifier;
  createdAt: string;
}

export interface LockParams {
  id: Identifier;
  identity: Identifier;
  meta?: Record<string, unknown>;
}

export interface UnlockParams {
  id: Identifier;
  identity: Identifier;
  meta?: Record<string, unknown>;
}

export interface GetLockParams {
  id: Identifier;
  meta?: Record<string, unknown>;
}

export interface GetLocksParams {
  meta?: Record<string, unknown>;
}

export class LockConflictError extends Error {
  readonly kind = "LockConflictError" as const;
  constructor(public readonly existingLock: Lock) {
    super(
      `Resource ${existingLock.resource}/${existingLock.recordId} is locked by ${existingLock.identity}`
    );
  }
}

export type RealtimeConnectionStatus =
  | "connected"
  | "reconnecting"
  | "disconnected"
  | "idle";

export type RealtimeTransportErrorKind =
  | "connect_failed"
  | "auth_failed"
  | "send_failed"
  | "parse_failed"
  | "handler_threw";

export interface RealtimeTransportError {
  kind: RealtimeTransportErrorKind;
  topic?: string;
  cause?: unknown;
  retrying: boolean;
}

export interface RealtimeTransport {
  subscribe<P = unknown>(
    topic: string,
    cb: SubscriptionCallback<P>
  ): Unsubscribe;
  publish<P = unknown>(
    topic: string,
    event: Omit<RealtimeEvent<P>, "topic">
  ): Promise<void>;
  connect?(): void;
  disconnect?(): void;
  onReconnect?(cb: () => void): Unsubscribe;
  onStatusChange?(cb: (status: RealtimeConnectionStatus) => void): Unsubscribe;
}

export interface LockProvider<R extends string = string> {
  lock(resource: R, params: LockParams): Promise<Lock>;
  unlock(resource: R, params: UnlockParams): Promise<Lock>;
  getLock(resource: R, params: GetLockParams): Promise<Lock | null>;
  getLocks(resource: R, params?: GetLocksParams): Promise<Lock[]>;
}

export interface RealtimeDataProvider<R extends string = string>
  extends DataProvider<R> {
  subscribe<P = unknown>(
    topic: string,
    cb: SubscriptionCallback<P>
  ): Unsubscribe;
  publish<P = unknown>(
    topic: string,
    event: Omit<RealtimeEvent<P>, "topic">
  ): Promise<void>;
  lock(resource: R, params: LockParams): Promise<Lock>;
  unlock(resource: R, params: UnlockParams): Promise<Lock>;
  getLock(resource: R, params: GetLockParams): Promise<Lock | null>;
  getLocks(resource: R, params?: GetLocksParams): Promise<Lock[]>;
  // Pass-through from transport so React hooks can subscribe without holding a transport ref.
  onReconnect?(cb: () => void): Unsubscribe;
  onStatusChange?(cb: (status: RealtimeConnectionStatus) => void): Unsubscribe;
}

export interface RealtimeDataProviderOptions<R extends string = string> {
  locks?: LockProvider<R>;
}
```

import type {
  RealtimeEvent,
  RealtimeTransport,
  RealtimeTransportError,
  SubscriptionCallback,
  Unsubscribe,
} from "../types";

export interface FakeTransportConfig {
  delayMs?: number;
  onError?: (error: RealtimeTransportError) => void;
}

export interface FakeTransport extends RealtimeTransport {
  simulateReconnect(): void;
  readonly publishedEvents: ReadonlyArray<{
    topic: string;
    event: Omit<RealtimeEvent, "topic">;
  }>;
}

export function fakeTransport(config: FakeTransportConfig = {}): FakeTransport {
  const subscribers = new Map<string, Set<SubscriptionCallback<unknown>>>();
  const reconnectListeners = new Set<() => void>();
  const events: { topic: string; event: Omit<RealtimeEvent, "topic"> }[] = [];

  const dispatch = (topic: string, event: Omit<RealtimeEvent, "topic">) => {
    const set = subscribers.get(topic);
    if (!set) return;
    const full: RealtimeEvent = { topic, ...event };
    for (const cb of set) {
      try {
        cb(full);
      } catch (cause) {
        config.onError?.({
          kind: "handler_threw",
          topic,
          cause,
          retrying: false,
        });
      }
    }
  };

  return {
    subscribe<P = unknown>(
      topic: string,
      cb: SubscriptionCallback<P>
    ): Unsubscribe {
      let set = subscribers.get(topic);
      if (!set) {
        set = new Set();
        subscribers.set(topic, set);
      }
      set.add(cb as SubscriptionCallback<unknown>);
      return () => {
        set!.delete(cb as SubscriptionCallback<unknown>);
        if (set!.size === 0) subscribers.delete(topic);
      };
    },

    async publish(topic, event) {
      events.push({ topic, event });
      if (config.delayMs && config.delayMs > 0) {
        await new Promise((r) => setTimeout(r, config.delayMs));
      }
      dispatch(topic, event);
    },

    onReconnect(cb) {
      reconnectListeners.add(cb);
      return () => {
        reconnectListeners.delete(cb);
      };
    },

    simulateReconnect() {
      for (const cb of reconnectListeners) cb();
    },

    get publishedEvents() {
      return events;
    },
  };
}
```

import type { DataProvider } from "ra-core";
import type {
  GetLockParams,
  GetLocksParams,
  LockParams,
  LockProvider,
  RealtimeDataProvider,
  RealtimeDataProviderOptions,
  RealtimeTransport,
  UnlockParams,
} from "./types";

export function realtimeDataProvider<R extends string = string>(
  baseDataProvider: DataProvider<R>,
  transport: RealtimeTransport,
  options: RealtimeDataProviderOptions<R> = {}
): RealtimeDataProvider<R> {
  const requireLocks = (method: string): LockProvider<R> => {
    if (!options.locks) {
      throw new Error(
        `realtimeDataProvider: ${method}() called but no LockProvider was configured. ` +
          `Pass { locks } in the third argument to enable locking.`
      );
    }
    return options.locks;
  };

  return {
    ...baseDataProvider,
    subscribe: (topic, cb) => transport.subscribe(topic, cb),
    publish: (topic, event) => transport.publish(topic, event),
    lock: (resource, params: LockParams) =>
      requireLocks("lock").lock(resource, params),
    unlock: (resource, params: UnlockParams) =>
      requireLocks("unlock").unlock(resource, params),
    getLock: (resource, params: GetLockParams) =>
      requireLocks("getLock").getLock(resource, params),
    getLocks: (resource, params?: GetLocksParams) =>
      requireLocks("getLocks").getLocks(resource, params),
    onReconnect: transport.onReconnect
      ? (cb) => transport.onReconnect!(cb)
      : undefined,
    onStatusChange: transport.onStatusChange
      ? (cb) => transport.onStatusChange!(cb)
      : undefined,
  };
}
```

import type { DataProvider } from "ra-core";
import { recordTopic, resourceTopic } from "./topics";
import type { RealtimeDataProvider } from "./types";

export function addEventsForMutations<R extends string = string>(
  baseDataProvider: DataProvider<R> | RealtimeDataProvider<R>,
  publisher: Pick<RealtimeDataProvider<R>, "publish">
): DataProvider<R> {
  return {
    ...baseDataProvider,

    create: async (resource, params) => {
      const result = await baseDataProvider.create(resource, params);
      await publisher.publish(resourceTopic(resource), {
        type: "created",
        payload: { ids: [result.data.id] },
      });
      return result;
    },

    update: async (resource, params) => {
      const result = await baseDataProvider.update(resource, params);
      await publisher.publish(recordTopic(resource, params.id), {
        type: "updated",
        payload: { id: params.id, data: result.data },
      });
      await publisher.publish(resourceTopic(resource), {
        type: "updated",
        payload: { ids: [params.id] },
      });
      return result;
    },

    updateMany: async (resource, params) => {
      const result = await baseDataProvider.updateMany(resource, params);
      await publisher.publish(resourceTopic(resource), {
        type: "updated",
        payload: { ids: result.data ?? params.ids },
      });
      return result;
    },

    delete: async (resource, params) => {
      const result = await baseDataProvider.delete(resource, params);
      await publisher.publish(recordTopic(resource, params.id), {
        type: "deleted",
        payload: { id: params.id, previousData: params.previousData },
      });
      await publisher.publish(resourceTopic(resource), {
        type: "deleted",
        payload: { ids: [params.id] },
      });
      return result;
    },

    deleteMany: async (resource, params) => {
      const result = await baseDataProvider.deleteMany(resource, params);
      await publisher.publish(resourceTopic(resource), {
        type: "deleted",
        payload: { ids: result.data ?? params.ids },
      });
      return result;
    },
  };
}

import { useEffect, useRef } from "react";
import { useDataProvider } from "ra-core";
import type {
  RealtimeDataProvider,
  SubscriptionCallback,
} from "../types";

export function useSubscribe<P = unknown>(
  topic: string,
  callback: SubscriptionCallback<P>,
  options?: { enabled?: boolean }
): void {
  const dataProvider = useDataProvider<RealtimeDataProvider>();
  const callbackRef = useRef(callback);
  callbackRef.current = callback;
  const enabled = options?.enabled !== false;

  useEffect(() => {
    if (!enabled) return;
    if (typeof dataProvider.subscribe !== "function") {
      throw new Error(
        "useSubscribe: dataProvider does not implement subscribe(). " +
          "Wrap your base dataProvider with realtimeDataProvider() first."
      );
    }
    const unsub = dataProvider.subscribe<P>(topic, (event) => {
      callbackRef.current(event);
    });
    return unsub;
  }, [dataProvider, topic, enabled]);
}

import { lockTopic, lockResourceTopic } from "../topics";
import {
  LockConflictError,
  type GetLockParams,
  type GetLocksParams,
  type Lock,
  type LockParams,
  type LockProvider,
  type RealtimeTransport,
  type UnlockParams,
} from "../types";

export interface InMemoryLockProviderOptions {
  publisher?: Pick<RealtimeTransport, "publish">;
}

const key = (resource: string, id: Lock["recordId"]) =>
  `${resource}::${String(id)}`;

export function inMemoryLockProvider<R extends string = string>(
  options: InMemoryLockProviderOptions = {}
): LockProvider<R> {
  const locks = new Map<string, Lock>();

  return {
    async lock(resource, params: LockParams) {
      const k = key(resource, params.id);
      const existing = locks.get(k);
      if (existing && existing.identity !== params.identity) {
        throw new LockConflictError(existing);
      }
      if (existing) return existing;
      const lock: Lock = {
        resource,
        recordId: params.id,
        identity: params.identity,
        createdAt: new Date().toISOString(),
      };
      locks.set(k, lock);
      await options.publisher?.publish(lockTopic(resource, params.id), {
        type: "locked",
        payload: lock,
      });
      return lock;
    },

    async unlock(resource, params: UnlockParams) {
      const k = key(resource, params.id);
      const existing = locks.get(k);
      if (!existing) {
        throw new Error(`No lock to release for ${resource}/${String(params.id)}`);
      }
      if (existing.identity !== params.identity) {
        throw new LockConflictError(existing);
      }
      locks.delete(k);
      await options.publisher?.publish(lockTopic(resource, params.id), {
        type: "unlocked",
        payload: existing,
      });
      // Also broadcast on the collection topic so list-level subscribers wake up.
      await options.publisher?.publish(lockResourceTopic(resource), {
        type: "unlocked",
        payload: existing,
      });
      return existing;
    },

    async getLock(resource, params: GetLockParams) {
      return locks.get(key(resource, params.id)) ?? null;
    },

    async getLocks(resource, _params?: GetLocksParams) {
      return [...locks.values()].filter((l) => l.resource === resource);
    },
  };
}

import { useCallback, useState } from "react";
import { useDataProvider, useGetIdentity } from "ra-core";
import type { Identifier } from "ra-core";
import type {
  Lock,
  LockParams,
  RealtimeDataProvider,
} from "../types";

export function useLock<R extends string = string>(): {
  lock: (
    resource: R,
    params: Omit<LockParams, "identity"> & { identity?: Identifier }
  ) => Promise<Lock>;
  isLoading: boolean;
  error: Error | null;
} {
  const dataProvider = useDataProvider<RealtimeDataProvider<R>>();
  const { identity: defaultIdentity } = useGetIdentity();
  const [isLoading, setIsLoading] = useState(false);
  const [error, setError] = useState<Error | null>(null);

  const lock = useCallback(
    async (
      resource: R,
      params: Omit<LockParams, "identity"> & { identity?: Identifier }
    ) => {
      const identity = params.identity ?? defaultIdentity?.id;
      if (identity == null) {
        throw new Error(
          "useLock: no identity provided and no authProvider identity available."
        );
      }
      setIsLoading(true);
      setError(null);
      try {
        const result = await dataProvider.lock(resource, {
          ...params,
          identity,
        });
        return result;
      } catch (e) {
        setError(e as Error);
        throw e;
      } finally {
        setIsLoading(false);
      }
    },
    [dataProvider, defaultIdentity]
  );

  return { lock, isLoading, error };
}

import { useEffect, useRef, useState } from "react";
import {
  useGetIdentity,
  useRecordContext,
  useResourceContext,
} from "ra-core";
import type { Identifier } from "ra-core";
import type { Lock } from "../types";
import { useLock } from "./use-lock";
import { useUnlock } from "./use-unlock";

export interface UseLockOnMountOptions {
  resource?: string;
  id?: Identifier;
  identity?: Identifier;
  meta?: Record<string, unknown>;
  onLockError?: (error: Error) => void;
}

export function useLockOnMount(
  options: UseLockOnMountOptions = {}
): { lock: Lock | null; isLocking: boolean; lockError: Error | null } {
  const ctxResource = useResourceContext();
  const ctxRecord = useRecordContext();
  const { identity: ctxIdentity } = useGetIdentity();
  const { lock } = useLock();
  const { unlock } = useUnlock();
  const heldRef = useRef<Lock | null>(null);

  const [held, setHeld] = useState<Lock | null>(null);
  const [error, setError] = useState<Error | null>(null);
  const [isLocking, setIsLocking] = useState(false);

  const resource = options.resource ?? ctxResource;
  const id = options.id ?? ctxRecord?.id;
  const identity = options.identity ?? ctxIdentity?.id;

  useEffect(() => {
    if (!resource || id == null || identity == null) return;
    let cancelled = false;
    setIsLocking(true);
    lock(resource, { id, identity, meta: options.meta })
      .then((result) => {
        if (cancelled) return;
        heldRef.current = result;
        setHeld(result);
      })
      .catch((e: Error) => {
        if (cancelled) return;
        setError(e);
        options.onLockError?.(e);
      })
      .finally(() => {
        if (!cancelled) setIsLocking(false);
      });

    return () => {
      cancelled = true;
      const heldLock = heldRef.current;
      if (heldLock) {
        // Best-effort unlock; ignore errors so we don't crash unmount.
        unlock(resource, { id, identity }).catch(options.onLockError);
        heldRef.current = null;
      }
    };
    // eslint-disable-next-line react-hooks/exhaustive-deps
  }, [resource, id, identity]);

  return { lock: held, isLocking, lockError: error };
}
```

import { Link, useLocation } from "react-router";
import { useStore } from "ra-core";
import { Badge } from "@/components/ui/badge";
import { resourceTopic } from "./topics";
import type { RealtimeEventType } from "./types";
import { useSubscribe } from "./hooks/use-subscribe";

export interface MenuLiveItemLinkProps {
  to: string;
  resource: string;
  primaryText: ReactNode;
  icon?: ReactNode;
  events?: RealtimeEventType[];
  badgeLabel?: (count: number) => ReactNode;
}

export function MenuLiveItemLink({
  to,
  resource,
  primaryText,
  icon,
  events = ["created"],
  badgeLabel,
}: MenuLiveItemLinkProps): ReactElement {
  const storeKey = `realtime.unread.${resource}`;
  const [count, setCount] = useStore<number>(storeKey, 0);
  const { pathname } = useLocation();

  useSubscribe(resourceTopic(resource), (event) => {
    if (events.includes(event.type)) setCount((c) => (c ?? 0) + 1);
  });

  if (pathname.startsWith(to) && count !== 0) {
    setCount(0);

  const label = badgeLabel

    : (count ?? 0) > 99
      ? "99+"
      : String(count ?? 0);

  return (

      {icon}
      <span>{primaryText}</span>
      {(count ?? 0) > 0 && (
        <Badge data-testid="menu-live-badge" variant="secondary">
          {label}
        </Badge>
      )}
    </Link>
  );
}

  items: ReadonlyArray<MenuLiveItemLinkProps>;

export function MenuLive({ items }: MenuLiveProps): ReactElement {
  return (
    <nav className="flex flex-col gap-1">
      {items.map((item) => (
        <MenuLiveItemLink key={item.to} {...item} />

  identity,
  meta,
  loading,
  lockedBy,

      </>
    );

  if (lock) return <>{children}</>;
  return <></>;
}

import type {
  RealtimeEvent,
  RealtimeTransport,
  SubscriptionCallback,
  Unsubscribe,
} from "../types";

export interface BroadcastChannelTransportConfig {
  channel: string;
}

interface WireMessage {
  topic: string;
  event: Omit<RealtimeEvent, "topic">;
}

export function broadcastChannelTransport(
  config: BroadcastChannelTransportConfig
): RealtimeTransport {
  let channel: BroadcastChannel | null = new BroadcastChannel(config.channel);
  const subscribers = new Map<string, Set<SubscriptionCallback<unknown>>>();

  const handleMessage = (msg: MessageEvent<WireMessage>) => {
    const { topic, event } = msg.data;
    const set = subscribers.get(topic);
    if (!set) return;
    const full: RealtimeEvent = { topic, ...event };
    for (const cb of set) {
      try {
        cb(full);
      } catch {
        // Isolated per spec §12; no onError hook for this transport in v1.
      }
    }
  };

  channel.addEventListener("message", handleMessage);

  return {
    subscribe<P = unknown>(
      topic: string,
      cb: SubscriptionCallback<P>
    ): Unsubscribe {
      let set = subscribers.get(topic);
      if (!set) {
        set = new Set();
        subscribers.set(topic, set);
      }
      set.add(cb as SubscriptionCallback<unknown>);
      return () => {
        set!.delete(cb as SubscriptionCallback<unknown>);
        if (set!.size === 0) subscribers.delete(topic);
      };
    },

    async publish(topic, event) {
      channel?.postMessage({ topic, event });
    },

    connect() {
      if (!channel) {
        channel = new BroadcastChannel(config.channel);
        channel.addEventListener("message", handleMessage);
      }
    },

    disconnect() {
      channel?.removeEventListener("message", handleMessage);
      channel?.close();
      channel = null;
    },
  };
}

import type {
  RealtimeEvent,
  RealtimeTransport,
  RealtimeTransportError,
  SubscriptionCallback,
  Unsubscribe,
} from "../types";

export interface WebSocketReconnectConfig {
  enabled?: boolean;
  initialDelayMs?: number;
  maxDelayMs?: number;
  maxAttempts?: number;
  jitter?: number;
}

export interface WebSocketTransportConfig {
  url: string | (() => string | Promise<string>);
  protocols?: string | string[];
  getAuthToken?: () => string | Promise<string>;
  authMode?: "query" | "subprotocol";
  reconnect?: WebSocketReconnectConfig;
  idleDisconnectMs?: number;
  heartbeatMs?: number;
  onError?: (error: RealtimeTransportError) => void;
}

interface OutgoingSubscribe { op: "subscribe"; topic: string; }
interface OutgoingUnsubscribe { op: "unsubscribe"; topic: string; }
interface OutgoingPublish { op: "publish"; topic: string; event: Omit<RealtimeEvent, "topic">; }
interface OutgoingPing { op: "ping"; }
type Outgoing = OutgoingSubscribe | OutgoingUnsubscribe | OutgoingPublish | OutgoingPing;

const DEFAULTS = {
  initialDelayMs: 1_000,
  maxDelayMs: 30_000,
  maxAttempts: Number.POSITIVE_INFINITY,
  jitter: 0.3,
  idleDisconnectMs: 30_000,
  heartbeatMs: 30_000,
};

export function webSocketTransport(
  config: WebSocketTransportConfig
): RealtimeTransport {
  const reconnect = { ...DEFAULTS, ...(config.reconnect ?? {}), enabled: config.reconnect?.enabled !== false };
  const idleDisconnectMs = config.idleDisconnectMs ?? DEFAULTS.idleDisconnectMs;
  const heartbeatMs = config.heartbeatMs ?? DEFAULTS.heartbeatMs;

  const subscribers = new Map<string, Set<SubscriptionCallback<unknown>>>();
  const reconnectListeners = new Set<() => void>();
  let socket: WebSocket | null = null;
  let connecting = false;
  let attempts = 0;
  let idleTimer: ReturnType<typeof setTimeout> | null = null;
  let heartbeatTimer: ReturnType<typeof setInterval> | null = null;
  let pongDeadline: ReturnType<typeof setTimeout> | null = null;
  let pending: Outgoing[] = [];

  const resolveUrl = async (): Promise<string> => {
    let url = typeof config.url === "function" ? await config.url() : config.url;
    if (config.getAuthToken && (config.authMode ?? "query") === "query") {
      const token = await config.getAuthToken();
      const u = new URL(url);
      u.searchParams.set("token", token);
      url = u.toString();
    }
    return url;
  };

  const send = (msg: Outgoing) => {
    if (socket?.readyState === WebSocket.OPEN) {
      try {
        socket.send(JSON.stringify(msg));
      } catch (cause) {
        config.onError?.({ kind: "send_failed", cause, retrying: false });
      }
    } else {
      pending.push(msg);
    }
  };

  const flushPending = () => {
    const queue = pending;
    pending = [];
    for (const msg of queue) send(msg);
  };

  const startHeartbeat = () => {
    if (heartbeatMs <= 0) return;
    heartbeatTimer = setInterval(() => {
      send({ op: "ping" });
      pongDeadline = setTimeout(() => {
        // No pong — force a reconnect.
        socket?.close(4000, "heartbeat timeout");
      }, Math.max(1_000, Math.floor(heartbeatMs / 3)));
    }, heartbeatMs);
  };

  const stopHeartbeat = () => {
    if (heartbeatTimer) clearInterval(heartbeatTimer);
    if (pongDeadline) clearTimeout(pongDeadline);
    heartbeatTimer = null;
    pongDeadline = null;
  };

  const scheduleReconnect = () => {
    if (!reconnect.enabled || attempts >= reconnect.maxAttempts) {
      config.onError?.({ kind: "connect_failed", retrying: false });
      return;
    }
    const base = Math.min(
      reconnect.initialDelayMs * 2 ** attempts,
      reconnect.maxDelayMs
    );
    const jitter = base * reconnect.jitter * (Math.random() * 2 - 1);
    const delay = Math.max(0, base + jitter);
    attempts += 1;
    setTimeout(connectInternal, delay);
  };

  const handleOpen = () => {
    attempts = 0;
    for (const topic of subscribers.keys()) {
      send({ op: "subscribe", topic });
    }
    flushPending();
    startHeartbeat();
    if (attempts === 0 && reconnectListeners.size > 0) {
      // First-time open isn't a reconnect; we only notify after a real reopen.
    }
  };

  const handleMessage = (ev: MessageEvent) => {
    let parsed: unknown;
    try {
      parsed = JSON.parse(String(ev.data));
    } catch (cause) {
      config.onError?.({ kind: "parse_failed", cause, retrying: true });
      return;
    }
    if (
      parsed &&
      typeof parsed === "object" &&
      "op" in parsed &&
      (parsed as { op?: unknown }).op === "pong"
    ) {
      if (pongDeadline) clearTimeout(pongDeadline);
      pongDeadline = null;
      return;
    }
    if (
      parsed &&
      typeof parsed === "object" &&
      "topic" in parsed &&
      "type" in parsed
    ) {
      const event = parsed as RealtimeEvent;
      const set = subscribers.get(event.topic);
      if (!set) return;
      for (const cb of set) {
        try {
          cb(event);
        } catch (cause) {
          config.onError?.({
            kind: "handler_threw",
            topic: event.topic,
            cause,
            retrying: false,
          });
        }
      }
    }
  };

  const handleClose = (ev: CloseEvent) => {
    stopHeartbeat();
    const wasOpen = socket?.readyState === WebSocket.CLOSED;
    socket = null;
    connecting = false;
    if (ev.code === 1000) return; // clean close
    scheduleReconnect();
    // After-reconnect listeners fire when handleOpen runs again.
    if (wasOpen) {
      for (const cb of reconnectListeners) cb();
    }
  };

  const handleError = (ev: Event) => {
    config.onError?.({ kind: "connect_failed", cause: ev, retrying: reconnect.enabled });
  };

  const connectInternal = async () => {
    if (connecting || socket) return;
    connecting = true;
    try {
      const url = await resolveUrl();
      const protocols =
        config.authMode === "subprotocol" && config.getAuthToken
          ? [await config.getAuthToken()]
          : config.protocols;
      socket = new WebSocket(url, protocols);
      socket.addEventListener("open", handleOpen);
      socket.addEventListener("message", handleMessage);
      socket.addEventListener("close", handleClose);
      socket.addEventListener("error", handleError);
    } catch (cause) {
      connecting = false;
      config.onError?.({ kind: "connect_failed", cause, retrying: reconnect.enabled });
      scheduleReconnect();
    }
  };

  const ensureConnected = () => {
    if (idleTimer) {
      clearTimeout(idleTimer);
      idleTimer = null;
    }
    if (!socket && !connecting) connectInternal();
  };

  const maybeIdleDisconnect = () => {
    if (subscribers.size > 0) return;
    if (idleDisconnectMs <= 0) return;
    idleTimer = setTimeout(() => {
      socket?.close(1000, "idle");
    }, idleDisconnectMs);
  };

  return {
    subscribe<P = unknown>(
      topic: string,
      cb: SubscriptionCallback<P>
    ): Unsubscribe {
      let set = subscribers.get(topic);
      if (!set) {
        set = new Set();
        subscribers.set(topic, set);
        send({ op: "subscribe", topic });
      }
      set.add(cb as SubscriptionCallback<unknown>);
      ensureConnected();
      return () => {
        set!.delete(cb as SubscriptionCallback<unknown>);
        if (set!.size === 0) {
          subscribers.delete(topic);
          send({ op: "unsubscribe", topic });
        }
        if (subscribers.size === 0) maybeIdleDisconnect();
      };
    },

    async publish(topic, event) {
      ensureConnected();
      send({ op: "publish", topic, event });
    },

    connect() {
      ensureConnected();
    },

    disconnect() {
      stopHeartbeat();
      socket?.close(1000, "client disconnect");
      socket = null;
    },

    onReconnect(cb) {
      reconnectListeners.add(cb);
      return () => {
        reconnectListeners.delete(cb);
      };
    },
  };
}
```

import type {
  RealtimeEvent,
  RealtimeTransport,
  RealtimeTransportError,
  SubscriptionCallback,
  Unsubscribe,
} from "../types";

export interface SSEReconnectConfig {
  enabled?: boolean;
  initialDelayMs?: number;
  maxDelayMs?: number;
  jitter?: number;
}

export interface SSETransportConfig {
  url: string | (() => string | Promise<string>);
  publishUrl?: string | (() => string | Promise<string>);
  getAuthToken?: () => string | Promise<string>;
  withCredentials?: boolean;
  reconnect?: SSEReconnectConfig;
  idleDisconnectMs?: number;
  topicFilterParam?: string;
  onError?: (error: RealtimeTransportError) => void;
}

const DEFAULTS = {
  initialDelayMs: 1_000,
  maxDelayMs: 30_000,
  jitter: 0.3,
  idleDisconnectMs: 30_000,
  topicFilterParam: "topics",
};

export function sseTransport(
  config: SSETransportConfig
): RealtimeTransport {
  const reconnect = {
    ...DEFAULTS,
    ...(config.reconnect ?? {}),
    enabled: config.reconnect?.enabled !== false,
  };
  const idleDisconnectMs = config.idleDisconnectMs ?? DEFAULTS.idleDisconnectMs;
  const topicFilterParam = config.topicFilterParam ?? DEFAULTS.topicFilterParam;

  const subscribers = new Map<string, Set<SubscriptionCallback<unknown>>>();
  const reconnectListeners = new Set<() => void>();
  let source: EventSource | null = null;
  let attempts = 0;
  let idleTimer: ReturnType<typeof setTimeout> | null = null;
  let everConnected = false;

  const resolveUrl = async (): Promise<string> => {
    let url = typeof config.url === "function" ? await config.url() : config.url;
    const u = new URL(url);
    const topics = [...subscribers.keys()].join(",");
    if (topics) u.searchParams.set(topicFilterParam, topics);
    if (config.getAuthToken) {
      const token = await config.getAuthToken();
      u.searchParams.set("token", token);
    }
    return u.toString();
  };

  const connectInternal = async () => {
    if (source) return;
    try {
      const url = await resolveUrl();
      source = new EventSource(url, {
        withCredentials: config.withCredentials,
      });
      source.onopen = () => {
        if (everConnected) {
          for (const cb of reconnectListeners) cb();
        }
        everConnected = true;
        attempts = 0;
      };
      source.onerror = () => {
        config.onError?.({
          kind: "connect_failed",
          cause: new Error("EventSource error"),
          retrying: reconnect.enabled,
        });
        source?.close();
        source = null;
        if (reconnect.enabled) scheduleReconnect();
      };
      // The server emits topic as the SSE event name.
      // Re-attach listeners every time we (re)connect.
      attachTopicListeners();
    } catch (cause) {
      config.onError?.({ kind: "connect_failed", cause, retrying: reconnect.enabled });
      if (reconnect.enabled) scheduleReconnect();
    }
  };

  const attachTopicListeners = () => {
    if (!source) return;
    for (const topic of subscribers.keys()) {
      source.addEventListener(topic, (ev: MessageEvent) => {
        let payload: Omit<RealtimeEvent, "topic">;
        try {
          payload = JSON.parse(ev.data);
        } catch (cause) {
          config.onError?.({ kind: "parse_failed", topic, cause, retrying: false });
          return;
        }
        const full: RealtimeEvent = { topic, ...payload };
        const set = subscribers.get(topic);
        if (!set) return;
        for (const cb of set) {
          try {
            cb(full);
          } catch (cause) {
            config.onError?.({
              kind: "handler_threw",
              topic,
              cause,
              retrying: false,
            });
          }
        }
      });
    }
  };

  const scheduleReconnect = () => {
    const base = Math.min(
      reconnect.initialDelayMs * 2 ** attempts,
      reconnect.maxDelayMs
    );
    const jitter = base * reconnect.jitter * (Math.random() * 2 - 1);
    const delay = Math.max(0, base + jitter);
    attempts += 1;
    setTimeout(connectInternal, delay);
  };

  const ensureConnected = () => {
    if (idleTimer) {
      clearTimeout(idleTimer);
      idleTimer = null;
    }
    if (!source) connectInternal();
  };

  const maybeIdleDisconnect = () => {
    if (subscribers.size > 0) return;
    if (idleDisconnectMs <= 0) return;
    idleTimer = setTimeout(() => {
      source?.close();
      source = null;
    }, idleDisconnectMs);
  };

  return {
    subscribe<P = unknown>(
      topic: string,
      cb: SubscriptionCallback<P>
    ): Unsubscribe {
      let set = subscribers.get(topic);
      const isNew = !set;
      if (!set) {
        set = new Set();
        subscribers.set(topic, set);
      }
      set.add(cb as SubscriptionCallback<unknown>);
      if (isNew) {
        // (Re)open the EventSource to include the new topic in the filter param.
        source?.close();
        source = null;
        ensureConnected();
      } else {
        ensureConnected();
      }
      return () => {
        set!.delete(cb as SubscriptionCallback<unknown>);
        if (set!.size === 0) subscribers.delete(topic);
        if (subscribers.size === 0) maybeIdleDisconnect();
      };
    },

    async publish(topic, event) {
      if (!config.publishUrl) {
        throw new Error("sseTransport: publish requires publishUrl in config");
      }
      const url =
        typeof config.publishUrl === "function"
          ? await config.publishUrl()
          : config.publishUrl;
      const headers: Record<string, string> = {
        "Content-Type": "application/json",
      };
      if (config.getAuthToken) {
        headers.Authorization = `Bearer ${await config.getAuthToken()}`;
      }
      const res = await fetch(url, {
        method: "POST",
        headers,
        body: JSON.stringify({ topic, event }),
        credentials: config.withCredentials ? "include" : "same-origin",
      });
      if (!res.ok) {
        const error: RealtimeTransportError = {
          kind: "send_failed",
          topic,
          cause: new Error(`Publish failed with HTTP ${res.status}`),
          retrying: false,
        };
        config.onError?.(error);
        throw new Error(`sseTransport.publish: HTTP ${res.status}`);
      }
    },

    connect() {
      ensureConnected();
    },

    disconnect() {
      source?.close();
      source = null;
      if (idleTimer) clearTimeout(idleTimer);
    },

    onReconnect(cb) {
      reconnectListeners.add(cb);
      return () => {
        reconnectListeners.delete(cb);
      };
    },
  };
}

**Files:**

- [ ] **Step 1: Write the overview page**

Create `docs/src/content/docs/Realtime.md`:

```md
# Realtime

The realtime subsystem extends `DataProvider` with subscribe / publish / lock
methods and ships hooks and components that turn any admin app into a live one.
Pick a transport (WebSocket, SSE, BroadcastChannel, or the in-memory fake),
wire it through `realtimeDataProvider()`, and swap your `<List>` / `<Edit>` /
`<Show>` for the live variants.

## Quick start

```tsx
import {
  realtimeDataProvider,
  webSocketTransport,
} from "@/components/realtime";
import baseDataProvider from "./my-rest-data-provider";

const transport = webSocketTransport({ url: "wss://example.com/realtime" });
const dataProvider = realtimeDataProvider(baseDataProvider, transport);
```

Then swap `<List>` → `<ListLive>`, `<Edit>` → `<EditLive>`, `<Show>` →
`<ShowLive>` where you want live updates.

## Pick a transport

| Transport | Direction | Cross-tab | Server needed | Use for |
|-----------|-----------|-----------|---------------|---------|
| `webSocketTransport` | bidirectional | yes | yes (WS server) | full realtime apps |
| `sseTransport` | server → client | yes | yes (SSE endpoint + optional publish endpoint) | dashboards, read-mostly apps |
| `broadcastChannelTransport` | bidirectional | yes (same browser) | no | small intranet apps + demos |
| `fakeTransport` | bidirectional | no | no | tests |

## Architecture

```
+----------------+    +------------------------+    +----------------+
| Your app code  | -> | realtimeDataProvider() | -> | RealtimeTransport
| (hooks, comps) |    | + optional LockProvider|    | (WS/SSE/etc)   |
+----------------+    +------------------------+    +----------------+
                                |
                                v
                          react-query cache
                          (invalidated by hooks)
```

## Topic conventions

- `resource/<name>` — collection updates
- `resource/<name>/<id>` — single-record updates
- `lock/<name>` — any lock event on the resource
- `lock/<name>/<id>` — lock event on a single record

Build them with `resourceTopic`, `recordTopic`, `lockResourceTopic`, `lockTopic`.

## CUD broadcast (`addEventsForMutations`)

Opt-in. Wrap your dataProvider so its own writes publish events back through the
transport:

```ts
import { realtimeDataProvider, addEventsForMutations } from "@/components/realtime";

const baseDP = realtimeDataProvider(...);
const dataProvider = addEventsForMutations(baseDP, baseDP);
```

Useful for cross-tab demos and single-user apps. Skip it if your backend already
broadcasts its own writes (Supabase, Pusher, Hasura) — otherwise events
duplicate.

## Locks

Locks are pluggable via the `LockProvider` interface. The kit ships
`inMemoryLockProvider()` for demos and tests; production apps implement their
own (talking to a REST endpoint, Redis, Postgres advisory locks, etc.). See the
per-hook pages for `<LockOnMount>`, `<LockStatus>`, `useLock`, `useUnlock`, and
the `useGet[s]Lock[s][Live]` family.

## Custom transports

Implement the `RealtimeTransport` interface:

```ts
export interface RealtimeTransport {
  subscribe(topic, cb): Unsubscribe;
  publish(topic, event): Promise<void>;
  connect?(): void;
  disconnect?(): void;
  onReconnect?(cb): Unsubscribe;
  onStatusChange?(cb): Unsubscribe;
}
```

See the WebSocket and SSE transports for reference implementations.

## Server protocol (WebSocket)

Clients send JSON frames with `op: "subscribe" | "unsubscribe" | "publish" | "ping"`.
Servers push `{ topic, type, payload }` frames or `{ op: "pong" }`. See the
`webSocketTransport` page for the full wire format.
```
