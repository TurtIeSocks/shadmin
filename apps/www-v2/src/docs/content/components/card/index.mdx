# Card

Container with header, content, and footer slots.

<ComponentPreview name="components/card/basic" />

## Usage

```tsx
import {
  Card,
  CardHeader,
  CardTitle,
  CardContent,
} from "shadmin/components/ui/card";

<Card>
  <CardHeader>
    <CardTitle>Card title</CardTitle>
  </CardHeader>
  <CardContent>Card body content here.</CardContent>
</Card>
```
