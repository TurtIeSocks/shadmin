# Alert

Informational banners with a title and description.

<ComponentPreview slug="components/alert" />

## Usage

```tsx
import {
  Alert,
  AlertTitle,
  AlertDescription,
} from "shadmin/components/ui/alert";

<Alert>
  <AlertTitle>Heads up!</AlertTitle>
  <AlertDescription>
    You can add components to your app using the CLI.
  </AlertDescription>
</Alert>
```

## Variants

`default` and `destructive`.
