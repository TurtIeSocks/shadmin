A real shadmin button — click me to trigger actions.

<ComponentPreview name="components/button/basic" />

## Usage

```tsx
import { Button } from "shadmin/components/ui/button";

<Button>Click me</Button>
<Button variant="outline">Outline</Button>
<Button variant="destructive">Delete</Button>
```

## Variants

The `Button` supports these variants: `default`, `outline`, `secondary`, `ghost`, `link`, `destructive`.
