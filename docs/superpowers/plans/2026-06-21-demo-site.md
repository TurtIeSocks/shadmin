# Demo Site Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a fresh single demo folded into `apps/www-v2` at `/demo` — a flagship admin + a docs-mirrored component gallery (live companion + visual pass-test) + feature demos, purple→indigo.

**Architecture:** Demo lives inside www-v2 as React-Router framework route modules under `/demo` (auto-code-split, so `ra-core`/`leaflet`/`admin` only load when `/demo` is opened). One `<CoreAdminContext>` + shadmin's real `<Layout>` with a custom 3-zone sidebar. 6 `ra-data-fakerest` resources. Gallery entries are glob-discovered files mirroring the docs content tree; a coverage radar flags any docs slug with no example.

**Tech Stack:** React Router 7 (framework mode, `ssr:false`), `shadmin-core` (ra-core re-export), `shadmin/components/admin/*` (source via alias), `ra-data-fakerest`, `data-generator-retail`, `react-shiki`, Tailwind v4, Biome, `node:test`.

## Global Constraints

- **Fold-in, not separate SPA:** all demo code lives under `apps/www-v2/src/demo/`; routes registered in `apps/www-v2/src/routes.ts` under `/demo`. Route modules auto-split — never add manual `lazy()`.
- **Never import `ra-core` directly.** Biome `noRestrictedImports` bans it in www-v2. Import every ra-core API (`CoreAdminContext`, `RecordContextProvider`, `ResourceContextProvider`, `useLogin`, `DataProvider` type, etc.) from **`shadmin-core`**. If a needed symbol isn't re-exported by shadmin-core, add it to shadmin-core's barrel in that task (do not override Biome).
- **Never import `radix-ui`/`@radix-ui/*` directly** — use `shadmin/components/ui/*`.
- **admin/leaflet from source; public API from dist.** Scoped Vite aliases map `shadmin/components/admin/*` and `shadmin/leaflet/*` to `packages/shadmin/src/...`. `shadmin/components/ui/*`, `shadmin/lib/*` keep resolving to dist via node_modules — do not alias them.
- **`@/` is importer-aware:** resolves to `packages/shadmin/src` when the importer is under `packages/shadmin/`, else `apps/www-v2/src`.
- **Exclude all `/demo` paths from prerender** (`react-router.config.ts`): demo is client-only ra-core. The existing `build` script already runs `copy-spa-fallback.mjs` for SPA fallback.
- **Tests:** `node:test` + `node:assert/strict`, pure-logic only (no DOM runner exists in www-v2). UI verified via typecheck + build + preview.
- **Path mirror:** `/demo/components/<cat>/<slug>` ↔ docs `/docs/<cat>/<slug>` exactly.
- **Theme:** purple→indigo. `--brand-from: #8b5cf6` → `--brand-to: #4f46e5` already in `apps/www-v2/src/index.css`. Set `--primary` to the brand violet on a `.theme-demo` wrapper.
- **Commands (from repo root):**
  - typecheck: `pnpm turbo run typecheck --filter=www-v2`
  - lint: `biome check apps/www-v2`
  - test: `pnpm turbo run test --filter=www-v2`
  - build: `pnpm turbo run build --filter=www-v2`
  - dev/preview: `pnpm turbo run dev --filter=www-v2` (port 5273)
- **Salvage, don't reinvent:** copy resource pages / seeds / providers from `apps/demo/src`, rewrite stale `shadmin/components/extras/*` imports to real `shadmin/components/admin/<subdir>/*` (mapping table in Appendix A).

// apps/www-v2/scripts/shadmin-source-resolver.mjs
import path from "node:path";

/** Which base dir `@/` should resolve against for a given importer. */
export function resolveAtImport(importer, { shadminSrc, wwwSrc }) {
  if (importer && path.resolve(importer).startsWith(path.resolve(shadminSrc))) {
    return shadminSrc;
  }
  return wwwSrc;
}

/**
 * Vite plugin: importer-aware `@/` (shadmin src vs www-v2 src) + scoped source
 * aliases for the dist-EXCLUDED parts of shadmin (admin, leaflet). The public
 * API (shadmin/components/ui/*, shadmin/lib/*) is intentionally NOT aliased so
 * it keeps resolving to the built dist via node_modules.
 */
export function shadminSourcePlugin({ shadminSrc, wwwSrc }) {
  const scoped = [
    { prefix: "shadmin/components/admin/", base: path.join(shadminSrc, "components/admin/") },
    { prefix: "shadmin/leaflet/", base: path.join(shadminSrc, "leaflet/") },
  ];
  return {
    name: "shadmin-source-resolver",
    enforce: "pre",
    async resolveId(source, importer) {
      if (source === "@" || source.startsWith("@/")) {
        const base = resolveAtImport(importer, { shadminSrc, wwwSrc });
        const rest = source === "@" ? "" : source.slice(2);
        return this.resolve(path.join(base, rest), importer, { skipSelf: true });
      }
      for (const { prefix, base } of scoped) {
        if (source.startsWith(prefix)) {
          return this.resolve(path.join(base, source.slice(prefix.length)), importer, { skipSelf: true });
        }
      }
      return null;
    },
  };
}
```

export function coverageReport(docSlugs: string[], exampleSlugs: string[]) {
  const have = new Set(exampleSlugs);
  return {
    covered: docSlugs.filter((s) => have.has(s)),
    missing: docSlugs.filter((s) => !have.has(s)),
  };
}
```
